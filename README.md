# 🎨 Pygame RGB Color Picker

An interactive RGB Color Picker built using **Python** and **Pygame**. This application allows users to adjust the **Red**, **Green**, and **Blue** color components using graphical sliders while displaying a real-time preview of the selected color.

---

## ✨ Features

- 🎨 Interactive RGB sliders
- 👀 Real-time color preview
- 🔴 Adjust Red, Green, and Blue values independently
- 🖥️ Live RGB values displayed in the window title
- ⚡ Lightweight and beginner-friendly
- 🐍 Built entirely with Python and Pygame

---

## 🛠️ Technologies Used

- Python 3.x
- Pygame

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/YourUsername/pygame-rgb-color-picker.git
```

### 2. Navigate to the project directory

```bash
cd pygame-rgb-color-picker
```

### 3. Install the required package

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python main.py
```

---

## 🎮 How to Use

1. Launch the application.
2. Use the mouse to move the sliders.
3. Adjust the Red, Green, and Blue components.
4. Observe the live color preview.
5. Read the RGB values directly from the window title.
